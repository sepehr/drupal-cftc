Description
===========
Enables the CCK Fieldgroup Tabs module to also include comments on a separate jQuery UI Tab.

Dependencies
============
- Comment Display (http://drupal.org/project/comment_display)
- CCK Fieldgroup Tabs (http://drupal.org/project/cck_fieldgroup_tabs)
